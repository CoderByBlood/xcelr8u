# User Stories

## Standard User

### Quality Attributes
Feature: As a user I want open and navigate the application with ease so that I can quickly get the value
- [ ] As a user I want the application to load and respond quickly so that I am not left wondering if it's working
- [ ] As a user I want the data to be encrypted at rest and in transit so that I feel safe using the application
- [ ] As a user I want the navigation to leverage the day planner metaphore for UI so that it is familar to me
- [ ] As a user I want a tutorial at application first open so that I can learn how to use it
- [ ] As a user I want a daily affirmation so that my spirits remain uplifted
- [ ] As a user I want a reward for capturing all of my one things for the day so that I stay motivated
- [ ] As a user I want a reward for completing all my habits for the day so that I stay motivated
- [ ] As a user I want a reward for completing all **Will Do** to-dos for the day so that I stay motivated
- [ ] As a user I want each habit to automatically carry over to today so that I do not have to re-enter them
- [ ] As a user I want each unfinished to-do to automatically carry over to today so that I do not have to re-enter them
- [ ] As a user I want to be able to export my data so that I can back it up
- [ ] As a user I want to be able to close my account so that I can no longer use the service

### Strategy Mountain
Feature: As a user I want to see my goal, milestones, principles, and valaues so that I can stay grounded in my strategy
- [ ] As a user I want to capture a goal and a set of milestones needed to achieve it so that I can clearly see big strategic win
- [ ] As a user I want to capture an ordered set of principles so that I can make trade-offs as I make decisions
- [ ] As a user I want to capture an unordered set of values so that I can remind myself of my uncompromising beliefs
- [ ] As a user I want to complete milestones so that I can mark my progress
- [ ] As a user I want to update the goal, milestones, principles, and values so that I can reflect clarity gained over time

### One Thing
Feature: As a user I want to start new day so that I can record emotional state and gratitude, as well as track habits and tasks.
- [ ] As a user I want to capture my emotional state everyday so that I can see trends over time
- [ ] As a user I want to capture the one thing I am grateful for everyday so that I start my day with positivity
- [ ] As a user I want to capture my one intention everyday so that I can clearly see the daily win
- [ ] As a user I want to capture the one thing to remember everyday so that I do not forget and writing it down helps to remeber
- [ ] As a user I want to capture the one highlight everyday so that I remember the good parts and be grateful for them
- [ ] As a user I want to capture the one learning everyday so that I record lessons and can go back and review them

### Habits
Feature: As a user I want to form new habits by associating new identies with those habits so that I can make long lasting changes
- [ ] As a user I want to add new habits and associated identies so that I can cleary see the change in order for the habit to stick
- [ ] As a user I want to edit existing habits and associated identies so that I can reflect clarity gained over time
- [ ] As a user I want to delate existing habits and associated identies so that I can remove ones that no longer serve me
- [ ] As a user I want to tag habits as either Wealth Generation, Loved Ones, or Self Care so that I can see where I'm focused
- [ ] As a user I want to complete one or more habits for a given day so that I an track trends over time

### Tasks
Feature: As a user I want to track non-habitual to-dos so that I can keep the commitments that I make to myself and others
- [ ] As a user I want to add a new to-do so that I can capure a new commitment
- [ ] As a user I want to update a to-do so that I can reflect clarity gained over time
- [ ] As a user I want to complete a to-do so that I can mark my progress
- [ ] As a user I want to delete a to-do so that I can reflect clarity gained over time
- [ ] As a user I want to tag to-dos as either Wealth Generation, Loved Ones, or Self Care so that I can see where I'm focused
- [ ] As a user I want to tag to-dos as either Will Do, Could Do, Won't Do so that I can focus on the important tasks

## Administrator
Feature: As an administrator I want maintin the health of the application and its user base
- [ ] As an administrator I want to create new users manually and automatically so that the penetration expands
- [ ] As an administrator I want to delete users manually and automatically so data is pruned after a user opts-out
- [ ] As an administrator I want to report on how the application is used so that I can see trends and make improvements
